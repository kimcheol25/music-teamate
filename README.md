김철- 블로그 홈페이지의 전반적인 구조를 기획하고 설계하여 메인 재즈 페이지와 세부 재즈 장르 페이지, 메인 발라드 페이지와 세부 발라드 장르 페이지를 각각 HTML로 구현하였다. 이 과정에서 홈페이지의 구성과 사용자 흐름을 지속적으로 개선하며 style.css와 script.js를 여러 차례 새롭게 작성·수정하여 디자인과 동작의 완성도를 높였다. 또한 각 HTML 파일에서 발생하는 구조적 오류와 기능적 문제를 개별적으로 분석하고 수정함으로써, 페이지 간 일관성과 안정적인 동작을 확보하였다.
----------------------------------------------------------------------------------------------------------------------------------
서민영-음악 블로그에서 HIPHOP, KPOP, POP, RNB 네 가지 장르에 대한 개별 페이지를 제작하고, 각 장르 페이지가 전체 사이트 디자인과 일관되게 보이도록 페이지 구조와 레이아웃을 설계했다. 또한 장르별 분위기와 특징을 고려해 추천 곡을 선정한 뒤, 곡 제목,아티스트,간단한 설명을 동일한 형식으로 정리하여 페이지에 배치했고, 각 장르의 느낌이 잘 드러나도록 스타일을 세부적으로 수정함과 동시에 공통 폰트와 색감, 구성 규칙을 유지해 사이트 전체 분위기와 자연스럽게 보이도록 완성했다.
-----------------------------------------------------------------------------------------------------------------------------------
김유정-트로트와 클래식 음악 장르를 기반으로 trot.html, classic.html을 생성하여 홈페이지를 제작하였다. 새로운 브랜치를 생성하여 해당 브랜치에서 작업을 수행하며 독립적인 페이지를 구성하였다. 홈페이지의 통일성과 가독성을 높이기 위해 팀원들이 사용한 스타일과 디자인을 바탕으로 하여 레이아웃과 디자인을 구성하였다.
